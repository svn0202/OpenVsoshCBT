# Матрица переноса функций

Статусы пересматриваются после изучения кода и тестирования.

| Возможность | TCExam 17 | Решение OpenVsoshCBT |
|---|---|---|
| PHP 8.2+, Composer, актуальные библиотеки | есть | использовать upstream |
| PostgreSQL и интеграционные тесты | есть | сделать PostgreSQL основным профилем |
| Ограниченное число повторений | есть с TCExam 15 | использовать upstream |
| Мобильный интерфейс экзамена | базовая адаптивность | **перенесено**: панель вопроса, крупные кнопки, сворачиваемый список, масштаб, тёмная тема и fullscreen |
| Отметка «Сомневаюсь» | нет | **перенесено локально** с ключом теста; серверное хранение требует отдельной модели данных |
| Авторазмер поля эссе | нет | **перенесено** без изменения формата ответа |
| Таймер вопроса и автопереход | есть в модели TCExam | проверить олимпиадный UX и крайние случаи |
| Русская локализация | базовая | расширять через TMX и `make lang` |
| Предварительная генерация вариантов | требует сравнения | кандидат на перенос с нагрузочными тестами |
| Аварийные офлайн-страницы | нет эквивалента TMFCBT | кандидат после модели целостности/импорта |
| Расширенные отчёты и Excel | частично | переносить отдельными экспортерами |
| Импорт XLSX пользователей/заданий | штатно TSV/XML | оценить, не тащить устаревший PHPExcel |
| Конвертер Word → XML | внешняя/неясная лицензия | не включать до чистой реализации и проверки прав |
| Общие настройки через UI | частично конфиги | добавлять только с проверкой прав и валидацией |
| Ограничение User-Agent/anti-tab | ненадёжно как защита | не считать механизмом безопасности |
| Автообновление TMFCBT | отсутствует | не переносить; обновления только через Git/релизы |
| Функции закрытой редакции | исходников нет | только независимая реализация публичных требований |

## Сверка с рабочей установкой

Эталон поведения: `/Users/nc/uris/серверыРЦОИ/tcexam-brest-redos8/local-app`,
TCExam 14.3.0 с накопленными TMFCBT- и локальными изменениями. Репозиторий
содержит рабочие незакоммиченные правки, поэтому сравнение выполняется без
изменения этой копии.

### Переносить следующими отдельными пакетами

| Пакет | Польза | Условие переноса |
|---|---|---|
| Настройки названия, логотипа и фона входа | привычное оформление площадки | хранить как валидируемую конфигурацию, не как сериализованный массив в `.json` |
| Статусы участников и безопасный сброс попытки | работа наблюдателя | явные права, POST + CSRF, журнал действий |
| Предварительная генерация вариантов | снижение пиковой нагрузки | транзакции, PostgreSQL/MySQL и нагрузочные тесты |
| Офлайн-листы и импорт ответов | аварийная работа без сети | подпись/контроль целостности, защита от повторного импорта |
| Дополнительные отчёты | организация олимпиады | чистый экспорт без старых TCPDF/PHPExcel |
| Импорт вопросов из Word | экономия времени методистов | новая реализация поверх поддерживаемых библиотек, проверка изображений и формул |

### Не переносить из legacy-кода напрямую

- AJAX/JSON-кэш вопросов в каталогах `qblock` и `answdata`: он дублирует
  состояние БД в доступных PHP-процессу файлах и усложняет согласованность;
- `tmf_update.php`: обновление приложения архивом поверх рабочей установки;
- встроенные копии CKEditor, PHPExcel, TCPDF и файлового менеджера;
- чат без отдельной модели авторизации и модерации;
- ограничения по User-Agent, блокировку вкладок и fullscreen как «защиту от
  списывания» — это допустимые организационные подсказки, но не граница
  безопасности;
- `tmf_general_settings.json` в текущем формате: несмотря на расширение,
  внутри находится URL-кодированный сериализованный PHP-массив.

## Критерии «хорошей» функции

Функция полезна, если решает реальную задачу олимпиады, не ухудшает
безопасность и доступность, работает на PostgreSQL, покрыта тестом, сохраняет
ответ при повторном запросе и допускает безопасный откат. Популярность функции
в другой сборке сама по себе не является критерием переноса.
