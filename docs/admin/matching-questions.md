# Вопросы на сопоставление

Тип `MATCHING` используется для установления взаимно-однозначных соответствий
между двумя наборами элементов.

## Создание вопроса

1. В редакторе вопроса выберите тип **Сопоставление**.
2. В тексте вопроса разместите пронумерованную левую колонку, например:

   ```text
   1. Россия
   2. Беларусь
   3. Казахстан
   ```

3. Создайте варианты правой колонки и укажите для каждого правильную позицию:

   | Вариант ответа | Позиция |
   |---|---:|
   | Москва | 1 |
   | Минск | 2 |
   | Астана | 3 |

4. Используйте позиции подряд, начиная с `1`. Позиции включённых вариантов не
   должны повторяться.

Участник выбирает номер соответствующего элемента возле каждого варианта.
Повторное назначение номера очищает его предыдущее назначение. Все элементы
управляются обычными HTML-списками выбора и доступны с клавиатуры.

## Баллы

При включённом частичном балле каждая пара оценивается отдельно, после чего
сумма делится на число пар. При выключенном частичном балле вопрос приносит
полный балл только при правильном сопоставлении всех элементов.

Перед публикацией проверьте полностью правильный ответ, одну переставленную
пару, пропущенную пару и попытку повторно использовать одну позицию.
