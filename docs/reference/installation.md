# Установка

TCExam 17 поддерживает Docker и ручную установку. Для production OpenVsoshCBT
рекомендуется воспроизводимый release-каталог, PostgreSQL, HTTPS и отдельные
рабочие конфиги.

## Docker для оценки

```sh
make up
```

Docker Compose запускает приложение и БД, выполняет CLI-установку и сохраняет
данные в volumes. Конфигурация по умолчанию предназначена для знакомства, а не
для олимпиады без дополнительной настройки и нагрузочного теста.

## Ручная установка

1. Установите PHP 8.2+, необходимые расширения, Composer, веб-сервер и БД.
2. Получите код релиза.
3. Выполните:

   ```sh
   composer install --no-dev --optimize-autoloader
   make lang
   ```

4. Дайте установщику временное право создать `admin/config`, `public/config`
   и `shared/config`; runtime-каталогам — постоянное минимальное право записи.
5. Запустите web-установщик либо `php install/install_cli.php` с переменными
   окружения.
6. Убедитесь, что создан уникальный `K_RANDOM_SECURITY`.
7. Закройте/удалите `install/`, смените стандартного администратора и включите
   HTTPS.
8. Выполните пробный тест и проверьте резервное восстановление.

Полные параметры CLI перечислены в заголовке `install/install_cli.php`.
Подробное руководство находится в корневом `install/README.md`.

## OpenVsoshCBT на РЕД ОС

Используйте [отдельное руководство](../admin/deployment-redos.md): оно
описывает PostgreSQL, права Apache, SELinux и размещение секретов.

## Обновление

Не устанавливайте новую версию поверх рабочего каталога. Разверните соседний
release, установите зависимости, сравните конфиги, примените миграции к копии
БД и выполните приёмку. См. [руководство обновления](../admin/upgrade.md).

Первоисточник: [Installation — TCExam](https://tcexam.org/docs/installation/).
