# Типы вопросов

TCExam 17 использует четыре базовых типа.

## MCSA — один вариант ответа

Участник выбирает один вариант. Правильность определяется флагом
`answer_isright` выбранного ответа. В корректно подготовленном вопросе должен
быть хотя бы один правильный вариант; для обычного MCSA оставляют один.

Подходит для классического одиночного выбора. Обязательно проверьте поведение
при отсутствии ответа и при случайном перемешивании вариантов.

## MCMA — несколько вариантов

Каждый вариант рассматривается как отдельное утверждение: участник либо
выбирает его, либо не выбирает. Правильным действием считается и выбор
правильного варианта, и отказ от выбора неправильного.

Отображение возможно флажками или группами переключателей в зависимости от
настроек. Итог может рассчитываться частично либо по правилу «всё или ничего».
Подробности: [расчёт баллов](scoring.md#mcma--несколько-вариантов).

## ORDER — упорядочивание

Участник задаёт позицию каждого элемента. Элемент считается правильным, если
выбранная позиция совпала с `answer_position`. Балл рассчитывается аналогично
MCMA: частично по элементам или целиком за весь вопрос.

Проверьте отсутствие повторяющихся позиций и удобство управления с клавиатуры.

## TEXT — свободный или краткий ответ

Ответом может быть слово, число, фраза или развёрнутый текст.

- Если у вопроса заданы включённые правильные короткие ответы, TCExam может
  оценить точное совпадение автоматически.
- Пробелы по краям удаляются.
- Сравнение может учитывать или не учитывать регистр в зависимости от
  `K_SHORT_ANSWERS_BINARY`.
- Если совпадения нет, балл остаётся неопределённым до ручной проверки.
- Развёрнутый ответ оценивает эксперт в форме проверки ответов.

Отдельно тестируйте значение `0`: оно является содержательным ответом и не
должно превращаться в «нет ответа».

## Общие свойства вопроса

У вопроса есть тема, описание, объяснение, тип, сложность, состояние,
позиция, таймер, полноэкранный режим, inline-отображение ответов и автопереход.
Доступность конкретных элементов зависит от типа и настроек теста.
