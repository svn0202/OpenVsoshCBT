# Изменения OpenVsoshCBT

## Первая итерация

- создан форк с полной историей официального TCExam;
- базовая версия — TCExam 17.2.4;
- `origin` указывает на `svn0202/OpenVsoshCBT`, `upstream` — на официальный
  `tecnickcom/tcexam`;
- проект получил имя OpenVsoshCBT в README и шаблонах интерфейса/почты;
- в интерфейс добавлена настраиваемая ссылка на исходный код развёрнутого
  релиза без удаления copyright и ссылок TCExam/Tecnick;
- добавлена русская документация для РЕД ОС и проведения олимпиад;
- перенесены 123 совместимых перевода из рабочей legacy-ветки: русская
  локализация дополнена для управления файлами, OMR, OTP, паролей тестов,
  порядка вопросов, сертификатов и системных сообщений;
- добавлен отдельный тип вопроса `MATCHING` («Сопоставление») на совместимой
  позиционной модели `ORDER`, с защитой от повторного назначения позиции;
- описан безопасный перенос функций TMFCBT вместо слияния старого дерева;
- зафиксирован принцип хранения секретов только в игнорируемых рабочих
  `*/config`;
- демонстрационный сертификат TCExam сохранён без изменений по решению проекта.
- внешняя карта документации заменена локальным русским справочником по
  описанию, возможностям, первому тесту, типам вопросов, расчёту баллов,
  безопасности, установке, базе данных и структуре кода.

## Что уже предоставляет база TCExam 17

- PHP 8.2–8.4 и Composer;
- PHPMailer 7 и tc-lib-pdf вместо встроенных устаревших копий;
- устранение legacy register-globals;
- исправления SQL injection, XSS, IDOR и более безопасная работа с файлами;
- HTML5 и улучшения WCAG 2.1 AA;
- PHPUnit и интеграционные тесты PostgreSQL/MySQL;
- Docker и неинтерактивная установка;
- отдельные `config.default` и игнорируемые рабочие конфиги;
- числовое ограничение повторных попыток;
- встроенная TMX-локализация и сборка `make lang`.

Изменения функциональности фиксируются здесь после реализации и приёмки, а не
в момент добавления идеи в roadmap.
