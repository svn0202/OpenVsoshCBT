# Матрица переноса функций

Статусы пересматриваются после изучения кода и тестирования.

| Возможность | TCExam 17 | Решение OpenVsoshCBT |
|---|---|---|
| PHP 8.2+, Composer, актуальные библиотеки | есть | использовать upstream |
| PostgreSQL и интеграционные тесты | есть | сделать PostgreSQL основным профилем |
| Ограниченное число повторений | есть с TCExam 15 | использовать upstream |
| Мобильная тема | есть тема Picoman | сначала проверить, не переносить старую тему целиком |
| Таймер вопроса и автопереход | есть в модели TCExam | проверить олимпиадный UX и крайние случаи |
| Русская локализация | базовая | расширять через TMX и `make lang` |
| Предварительная генерация вариантов | требует сравнения | кандидат на перенос с нагрузочными тестами |
| Аварийные офлайн-страницы | нет эквивалента TMFCBT | кандидат после модели целостности/импорта |
| Расширенные отчёты и Excel | частично | переносить отдельными экспортерами |
| Импорт XLSX пользователей/заданий | штатно TSV/XML | оценить, не тащить устаревший PHPExcel |
| Конвертер Word → XML | внешняя/неясная лицензия | не включать до чистой реализации и проверки прав |
| Общие настройки через UI | частично конфиги | добавлять только с проверкой прав и валидацией |
| Ограничение User-Agent/anti-tab | ненадёжно как защита | не считать механизмом безопасности |
| Автообновление TMFCBT | отсутствует | не переносить; обновления только через Git/релизы |
| Функции закрытой редакции | исходников нет | только независимая реализация публичных требований |

## Критерии «хорошей» функции

Функция полезна, если решает реальную задачу олимпиады, не ухудшает
безопасность и доступность, работает на PostgreSQL, покрыта тестом, сохраняет
ответ при повторном запросе и допускает безопасный откат. Популярность функции
в другой сборке сама по себе не является критерием переноса.
